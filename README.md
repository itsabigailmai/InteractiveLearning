## Getting Started

### Prerequisites

- Node.js 18+ installed
- OpenAI API key
- ElevenLabs API key

### Installation

1. Clone the repository and navigate to the project:

```bash
cd interactive-learning
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env.local` file in the root directory:

```env
OPENAI_API_KEY=your_openai_api_key_here
ELEVENLABS_API_KEY=your_elevenlabs_api_key_here
NEXT_PUBLIC_DEMO_PASSWORD=learningisfun123
```

4. Run the development server:

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

### Login Credentials

For the demo, you can use any email address and the password: `learningisfun123`

## Content Library

The platform includes curated lessons across three grade levels:

## Deployment

### Deploy to Vercel

1. Push your code to GitHub

2. Import your repository to Vercel

3. Add environment variables in Vercel dashboard:
   - `OPENAI_API_KEY`
   - `ELEVENLABS_API_KEY`
   - `NEXT_PUBLIC_DEMO_PASSWORD`

4. Deploy!

## Design Philosophy

This platform is designed to be **playful, colorful, and non-professional** to appeal to middle school kids and below. The design features:

- Bright, vibrant colors (purple, yellow, blue, pink)
- Rounded corners and playful animations
- Friendly fonts (Poppins)
- Fun emojis and icons throughout
- Encouraging, positive language
- Smooth transitions and hover effects
