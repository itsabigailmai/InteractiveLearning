# IMPORTANT: Create a .env.local file

Copy the contents below and create a file called `.env.local` in the `interactive-learning` directory:

```
OPENAI_API_KEY=your_openai_api_key_here
ELEVENLABS_API_KEY=your_elevenlabs_api_key_here
NEXT_PUBLIC_DEMO_PASSWORD=learningisfun123
```

Replace the API keys with your actual keys from:
- OpenAI: https://platform.openai.com/api-keys
- ElevenLabs: https://elevenlabs.io/app/settings/api-keys

The demo password is set to `learningisfun123` by default.
